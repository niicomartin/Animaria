ANIMARIA - VERSIÓN PREPARADA PARA PUBLICAR
==========================================

Este ZIP quedó preparado para trabajar en local y también para publicar en internet.

QUÉ SE MODIFICÓ
===============

1) application.yml
   - Ahora usa variables de entorno.
   - En tu PC sigue funcionando con localhost, root/root y puerto 8080.
   - En Render/Railway usa SPRING_DATASOURCE_URL, SPRING_DATASOURCE_USERNAME y SPRING_DATASOURCE_PASSWORD.

2) Dockerfile
   - Permite publicar el proyecto como contenedor.
   - Render puede usarlo directamente si elegís entorno Docker.

3) render.yaml
   - Archivo de blueprint para Render ubicado en la raíz del ZIP.
   - Define el servicio web Animaria con rootDir PetShop.

4) Correo
   - Preparado para Gmail con contraseña de aplicación.
   - Los clientes pueden registrarse con cualquier correo.
   - El correo emisor tiene que estar en MAIL_USERNAME y MAIL_PASSWORD.

5) Admin
   - El email administrador sigue siendo forrajelahuella@hotmail.com.
   - Todos los demás usuarios quedan GENERAL.
   - Para limpiar usuarios de prueba, usar LIMPIAR_USUARIOS=true una vez o ejecutar el SQL incluido.
